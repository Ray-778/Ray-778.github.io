python之字符串前面加u,r,f的含义

发表于 2021-01-08 分类于 编程学习 ， python

版本 python3.6及以上

1.u:表示unicode字符串
字符串中存在中文的字符，在前面加上u

2.r:表示非转义的原始字符串
字符串中存在\，要让它失效，要在前面加上r

3.f:表示在字符串内支持大括号内的python 表达式

f-strings 是指以 f 或 F 开头的字符串，其中以 {} 包含的表达式会进行值替换。（目前支持python3.6版本）
